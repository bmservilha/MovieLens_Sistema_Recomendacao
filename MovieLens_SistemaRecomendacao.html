Análise exploratória de dados e Sistema de Recomendação para o dataset MovieLens¶

O dataset MovieLens composto por 100004 ratings e 1296 tags aplicadas a 9125 filmes, distribuidos em 671 usurios. Este dataset largamente utilizado na construo e validao de sistemas de recomendao. Neste projeto uma nalise exploratria dos dados contidos nesse dataset ser realizada de modo a responder as seguintes perguntas:

  • Existe alguma relao entre a avaliao e o gnero cinematogrfico?
  • Quais gneros possuem maior correlao entre si

A seguir, um sistema de recomendao ser construdo de maneira a confirmar as concluses das questes acima.

As similaridades entre os filmes e entre os usurios calculada utilzando a mtrica de similaridade entre cossenos e so armazendas nas matrizes movie_similarity e user_similarity, respectivamente. A matriz dos filmes obtida aplicando-se diretamente a funo pairwise_distance e a matriz dos usurios obtida aplicando essa funo na matriz transposta contendo as features. Features prximas levaro valores de cosseno mais elevados. Essa mtrica mais recomendada do que a distncia euclideana, por exemplo, uma vez que a vetorizao das features leva a melhores orientaes para o sistema de recomendao

A funo de similaridade entre cossenos dada por:

$$ sim(a,b) = cos(\pmb x, \pmb y) = \frac {\pmb x \cdot \pmb y}{||\pmb x|| \cdot ||\pmb y||} $$

user_similarity = 1 - pairwise_distances(ratings_norm.T, metric = 'cosine')

A funo getUserRecommendation leva o userId como parmetro e gera uma lista de usurios similares ao usurio do argumento, em ordem decrescente. A funo retorna um dataframe contendo como ndice o movieId do filme e ttulo e gneros em que o filme classificado.

def getUserBasedRecommendation(userId):
    '''
    parmetro: userId.
    
    Retorna um dataframe contendo os 10 filmes recomendados para o userId, baseando-se na ordenao entre as similaridades
    dos filmes com o usurio.
    
    
    '''
    # pega o ndice do filme do dataframe movies_df
        
    # Calcula a diferena entre as similares do movieName com e ordena em uma lista
    sim_scores = list(enumerate(movie_similarity[userId]))

    # ordenao da lista
    sim_scores = sorted(sim_scores, key=lambda x: x[1], reverse=True)

    # Seleciona os dez filmes com scores mais prximas, comeando do ndice 1,j que o ndice 0  o prprio filme!
    sim_scores = sim_scores[1:12]

    # pega o ndice dos animes mais prximos
    userId_indices = [i[0] for i in sim_scores]

    # devolve os dez filmes mais similares ao movieName, varrendo a lista userId_indices
    movies = []
    for i in userId_indices:
        result = movies_df[['title', 'genres']].iloc[userId_indices].drop(i)
        movies.append(result)
    return movies[0]

2.1 Validação¶

De modo a validar o sistema de recomendao proposto, criou-se uma funo de predio, contendo o ttulo de filme e um userId como parmetros, que retorna a mdia ponderada das similaridades entre as avaliaes para o titulo, tendo como peso a similaridade entre os usurios, matematicamente, temos: $$ pred = \frac{\sum_{i=1}^{671} rating * similarity}{\sum_{i=1}^{671} similarity} $$

def predict(movieId, userId):
    '''
    Parmetros: movieId e userId
    
    Retorna a mdia ponderada das avaliaes pela similaridade.
       
    '''
    # lista em ordem decrescente de usurios similares, lista, tambm em ordem decrescente com os
    # valores contendo as similaridades
    similarUser = user_similarity_df.sort_values(by=userId, ascending=False).index[1:1000]
    userValues = user_similarity_df.sort_values(by=userId, ascending=False).loc[:,userId].tolist()[1:1000]
    
    rating_list = []
    weight_list = []
    for j, i in enumerate(similarUser):
        rating = user_similarity_df.loc[i, movieId]
        similarity = userValues[j]
        rating_list.append(rating*similarity)
        weight_list.append(similarity)
    return sum(rating_list)/sum(weight_list)
